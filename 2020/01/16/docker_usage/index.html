基本概念

  • 镜像(image)
  • 容器(container)

粗浅的理解: 镜像相当于一个类, 容器就是根据这个类创建的对象。详细参考Docker从入门到实践

镜像又分为远程镜像和本地镜像。这两个和git类似, 一个是远程, 一个是本地。
